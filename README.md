# Dijkstra's Shortest Path Algorithm 🚀

## 📌 Overview

This project implements Dijkstra’s algorithm to compute the shortest path in a graph. It includes multiple optimized and extended versions of the algorithm for better performance and analysis.

---

## ⚙️ Features

* Standard Dijkstra algorithm
* Early stopping optimization for faster destination search
* Path reconstruction from source to destination
* Landmark-based distance computation
* Dijkstra with potential-based edge reweighting
* Modified version using maximum potential selection
* Visualization of shortest path

---

## 🧠 Concepts Used

* Graph algorithms
* Greedy approach
* Shortest path optimization
* Path reconstruction using parent array
* Edge reweighting techniques

---

## 🛠️ Technologies Used

* C Programming

---

## 📊 Sample Output

* Displays shortest distances from source node
* Shows shortest path from source to destination
* Demonstrates optimized algorithm variations
